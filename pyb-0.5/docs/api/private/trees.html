Module Hierarchy

  • pyb: Main pyb package.
    • core: Core classes for pyb.
    • filters: Module which contains a bunch of useful regular expression filters.
    • tasks: Task module which contains all the core tasks for Pyb.
      • java: Tasks related to Java.
      • jython: Tasks related to Jython.
      • py: Python related tasks.
    • util: Utility functions.

Class Hierarchy

  • __builtin__.object: The most base type
    • unittest.TestCase: A class whose instances are single test cases.
      • pyb.util._test: Unit tests for this module
    • __builtin__.type: type(object) -> the object's type type(name, bases, dict) -> a new type
  • exceptions.Exception: Common base class for all exceptions.
    • pyb.core.BuildException: Raised if an error occurs during the build.
    • exceptions.StandardError: Base class for all standard Python exceptions.
      • exceptions.AssertionError: Assertion failed.
  • pyb.core.Project: Class representing a complete project.
  • pyb.core.Target: Class representing a single build target.
  • pyb.core._MethodWrapper: Wrap the target body method allowing pre- and post- method code.
